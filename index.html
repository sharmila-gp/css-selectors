SIMPLE SELECTOR
Element Selector Tag name selects HTML elements based on the element name P selects all the P tag in the html page Example
Class Selector class name . class selector selects HTML elements with a specific class attribute. .test select the element which has class name as test. Example
Id Selector id name # id selector is used to select one unique element #id1 select the element which has id1. Example
Universal Selector * select all the html element on the page *{ } Apply the style for the element on the page Example
Grouping Selector tag name,tag name,tag name apply style for all specified tag names table,tr,td select table, td, tr element Example
COMBINATOR SELECTOR
Descendant Selector Tag name tag name " " selects all the specified tag inside the specified parent element div p selects the p tag inside the div tag Example
Child Selector tag name > tag name > selects all the specified direct child element inside parent element div > p selects all the direct p tag inside the div tag. Example
Adjacent Sibling Selector tag name + tag name + select all the element that is directly or immediately after the specified element. div + p select all the p tag next to div tag. Example
General Sibling Selector tag name ~ tag name ~ select the element are next siblings of a specified element. div ~ p select the p tag next of a div tag. Example

PSUEDO-ELEMENT SELECTOR
first-line tag name :: first-line :: used to add a special style to the first line of a text. p::first-line select the first-line of the p tag add style to it. Example
first-letter tag name :: first-letter :: used to add a special style to the first letter of a text. p::first-letter select the first-letter of the p tag Example
before tag name :: before :: used to insert some content before the content of an element. img::before create a element before the img tag Example
after tag name :: after :: used to insert some content after the content of an element. img::after create a element after the img tag Example
marker :: marker :: selects the markers of list items.used in li tag. ::marker select the list round in ul tag Example
selection :: selection :: matches the portion of an element that is selected by a user. p::selection matches the portion of P tag that is selected by a user. Example
ATTRIBUTE SELECTOR
[attribute] selector tag name[attribute] used to select elements with a specified attribute. a[target] select a tag with attribute target. Example
[attribute="value"] selector tag name[attribute="value"] used to select elements with a specified attribute and value. a[target="_blank"] select a tag with attribute target="_blank". Example
[attribute~="value"] selector [attribute~="value"] ~ used to select elements with an attribute value containing a specified word. [title~="flower"] select the attribute with [title="flower"] Example
[attribute|="value"] selector [attribute|="value"] | used to select elements with the specified attribute, whose value can be exactly the specified value, or the specified value followed by a hyphen (-). [class|="top"] select the attribute with class that starts with the top before (-). Example
[attribute^="value"] selector [attribute^="value"] ^ used to select elements with the specified attribute, whose value starts with the specified value. [class^="top"] select the element that start with class=top Example
[attribute$="value"] Selector [attribute$="value"] $ used to select elements whose attribute value ends with a specified value. [class$="test"] select the element that end with the test attribute Example
[attribute*="value"] Selector [attribute*="value"] * used to select elements whose attribute value contains a specified value. class*="top" select all the element which has class name top Example
